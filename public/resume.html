01 AIBOO — Edge AI 음성 비서 (Jetson Orin) 2025.02 ~ 현재 · 단독 개발
공공기관(보훈부) 노인 케어 사업용 음성 비서. xiaozhi-esp32-server fork를 단일 Python 프로세스로 리팩토링 — 메모리 약 750MB 절감, PM2로 서비스 3개 분리해 장애 격리 유지. Jetson Orin은 CTranslate2 CUDA 버전 맞춰 소스 빌드. 도메인 기능 8종(센서·건강·스케줄·긴급·정서지원·i18n 등) 구현, ESP32 펌웨어 빔포밍(XVF3800) 설정, Barge-in은 TTS cut 방식으로 구현.
스택: ESP-IDF · MQTT · XVF3800
02 Multi-NPU 모델 포팅 (Kneron KL630 + Rockchip RK3588) 2025.02 ~ 09
서로 다른 NPU 아키텍처에 객체 탐지 모델 포팅. Kneron Toolchain(ktc) 양자화·컴파일, Rockchip RKNN INT8 변환. NPU 미지원 연산은 CPU에서 후처리로 분리.
스택: Kneron Toolchain · RKNN Toolkit
03 Qwen LoRA 파인튜닝 + 합성 데이터 파이프라인 2025.12 ~ 현재 · AIBOO LLM 모듈
외부 API 사용 불가 환경 → ollama 기반 로컬 LLM 샘플링으로 합성 데이터 생성. PEFT(LoRA) + TRL(SFTTrainer)로 Qwen 4가지 크기(0.5B / 1.5B / 3-1.7B / 3-4B) 학습, HuggingFace → GGUF Q4_K_M → Ollama 변환 스크립트 자동화.
스택: PEFT · TRL · llama.cpp
04 Fire/PPE Detection + RTSP 모니터링 2025.02 ~ 09
YOLOv5s 6클래스 학습 mAP@0.5 84.7%, RTSP 카메라 연동 Streamlit 실시간 모니터링.
스택: OpenCV · Streamlit
